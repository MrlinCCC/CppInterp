# CppInterp

**CppInterp** 是一个用 C++ 编写的解释器项目，旨在学习和实践编译原理中的核心环节，例如：
- 词法分析 (Lexical Analysis)
- 语法分析 (Parsing)
- 抽象语法树 (AST) 构建
- 表达式求值 (Evaluation)

## 技术栈

- C++20：现代 C++标准，提升代码性能与安全性
- CMake：跨平台构建工具，简化项目构建配置
- Vcpkg：C++包管理工具，简化依赖库安装
- Git：版本控制系统，方便团队协作与代码管理
- Google Test：单元测试框架，保障代码质量
